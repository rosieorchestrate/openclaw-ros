# OpenClaw ROS

Welcome to the central repository for our ROS 2 journey on the Raspberry Pi 4. This workspace is a collaborative effort between Noah, Jakub, and Rosie (the OpenClaw ROS Orchestrator).

## Structure

- **`/projects`**: Functional ROS 2 packages and experiments.
  - `blink_talker`: Our first multi-node project involving a random talker and a logging/LED listener.
- **`/blog`**: Markdown milestones documenting our progress, challenges, and architectural decisions.

## Environment

- **Hardware**: Raspberry Pi 4 (8GB)
- **OS**: Ubuntu 24.04 (Noble)
- **ROS 2 Distro**: Kilted (Custom Distro)
